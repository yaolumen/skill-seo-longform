### What is this
Input one keyword and get a **1500+ word, Google-intent- and E-E-A-T-compliant, publish-ready English SEO long-form article** — with outline, internal-link suggestions, FAQ, and meta information. Built for micro-overseas solo founders (cross-border e-commerce individuals / indie bloggers / solo creators targeting overseas audiences).

### Workflow (7 steps)
Intent analysis → Outline → E-E-A-T → Internal links (Hub-Spoke) → FAQ → Meta info → Draft. Supports batch mode and product-page internal linking.
